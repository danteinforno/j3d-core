Instructions for downloading and building j3d-core

To build the Java 3D packages, you must checkout the following three CVS repositories:

  • j3d-core
  • j3d-core-utils
  • vecmath

These three top-level directories must be named exactly as shown above and they must be sibling directories. To ensure this, run the cvs checkout command for each of the respositories from the same parent directory. For example:

    cd <cvs-root-dir>
    cvs checkout vecmath
    cvs checkout j3d-core
    cvs checkout j3d-core-utils

After you have downloaded the three respositories, follow the instructions in the README-build.html files in the "vecmath" and "j3d-core" directories. If you wish to browse the README.html file on-line before you checkout the sources, you can browse the CVS sources and click on the latest revision of README-build.html.
